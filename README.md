# AlloySphere

AlloySphere is a Startup Operating System that enables founders, talent, and investors to discover opportunities, form teams, execute projects, track startup growth, and collaborate within a unified platform.

## Documentation Overview

This repository is fully prepared for local development and production deployment. Please refer to the following documentation files for specific workflows:

1. **[PROJECT_SETUP.md](./PROJECT_SETUP.md)**: Instructions for checking out the codebase, installing dependencies, and running the local development server.
2. **[ENVIRONMENT_VARIABLES.md](./ENVIRONMENT_VARIABLES.md)**: A complete map of all required and optional environment variables.
3. **[SUPABASE_SETUP.md](./SUPABASE_SETUP.md)**: Guide to provisioning and configuring the Supabase database, storage, and authentication.
4. **[GOOGLE_AUTH_SETUP.md](./GOOGLE_AUTH_SETUP.md)**: Instructions for configuring Google Cloud Console OAuth credentials.
5. **[DEPLOYMENT.md](./DEPLOYMENT.md)**: Step-by-step guide to deploying AlloySphere to Vercel.

## Core Stack

- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Database & Auth**: Supabase (PostgreSQL)
- **Deployment**: Vercel

## Key Features

- **Founder Workflows**: Roadmap execution, Team management, Workspace organization.
- **Talent Discovery**: Startup discovery, job board, application pipeline.
- **Investor Tracking**: Deal flow, startup velocity monitoring, public roadmaps.
- **Community Feed**: Media-rich startup updates, announcements, and document sharing.
